# Contributing 

A rare specimen *(noun)*: An individual willing to improve the Farix source code. If you are indeed one of these, please do note that upon opening a Pull Request, submitting a patch, or contributing code to this repository:-

1. **Code:** The code you provide must pass through the linter. Documentation specifies required coding style.
2. **CLA Compliance:** You must strictly adhere to all terms, contribution standards, and licensing protocols defined in our [Contributor License Agreement](./legal/CLA.md).
